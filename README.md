# Aquagas  Backend node.js +MySQL
This is the backend API for Aquagas Delivery System poering Flutter apps user rider vendor and the admin panel

# Tech Stack
- Node.js
- Express.js
- Mysql Railway / Planetscale

# Setup
1. clone repo
2. Install dependencies: npm install
3. Create .env file
4. Start server: npm run dev

# Deployment
